# Iframe

Allow to embed iframes (embed websites) with title and description

## Fields

| Field Name  |        Type         | Required | Details                                                      |
| :---------- | :-----------------: | :------: | :----------------------------------------------------------- |
| Title	 | Text (formatted) |   No    |   |
| Description |      Text (formatted, long)	|    No    |           |
| Iframe URL	 | Text (plain) |   Yes    |   |
